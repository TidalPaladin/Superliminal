Dropbox Help

Linking An Account

Clicking the 'Link Dropbox' button will open Dropbox.com in a new tab where you will be asked to authorize Superliminal. Copy the code given by Dropbox and paste it into the 'Paste authentication here' box. Give a name to this linked account in the box above. Multiple accounts can be linked, allowing for easy deployment of a single image to multiple locations

  • Account Name

    Name by which Superliminal will remember this Dropbox account. This is only visible to Superliminal, and is used to identify which Dropbox token to use when fetching images.

  • Dropbox Token

    Paste the access token given by dropbox.com into this field. The token will be generated after you link Superliminal to your Dropbox on Dropbox.com

Manage Accounts

  • Download Accounts File

    Downloads a file containing a master list of all linked Dropbox accounts.

  • Upload Accounts File

    Uploads a master list of Dropbox accounts (accounts.json). Uploading this to a second copy of Superliminanl will import all Dropbox accounts from the first image.

  • Use Config

    The Dropbox account to fetch flyers from. The list of names given here match the name given under 'Account Name' when the Dropbox account was first added.

  • Delete Config

    Chooses a Dropbox config file to delete. As of now, deleting the config file does not unlink Superliminal from your Dropbox account. If necessary, Superliminal can be ulinked from your Dropbox account at Dropbox.com
