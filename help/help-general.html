Superliminal

General Help

Login Settings

  • Login

    Username used to log in to the Superliminal configuration page

  • Password

    Password used to log in to the Superliminal configuration page

Hotspot Settings

  • Hotspot

    If Superliminal is configured to run in network (non-USB) mode but no network connection is available, Superliminal will host a hotspot. Administrators can connect to this hotspot and reconfigure network settings without needing to physically access hardware with a mouse and keyboard.

  • SSID

    The name (ESSID) of the hotspot

  • Passphrase

    The WPA2 passphrase of the hotspot. This must be at least 8 alphanumeric charaters long

System Settings

  • Mode

    In USB mode flyers are fetched from a USB drive at boot. In Network mode flyers are fetched from a dropbox account over the network.

  • Dropbox Account

    Name of the Dropbox account to fetch flyers from. This is the name given when the account was first added.

  • Max Wait for Network

    The maximum time in seconds to wait for network connectivity. Only applicable when in 'Network' mode. If the network is still down at the end of this interavl, the slideshow will launch using flyers from the last successful run.

  • Max Wait for USB

    The maximum time in seconds to wait for a USB drive to be inserted. Only applicable when in 'USB' mode. If no USB drive has been inserted at the end of this interval, the slideshow will launch using flyers from the last successful run.

  • Slideshow Interval

    The time in seconds to show a flyer before transitioning to the next.

  • Delay for TV

    The time in seconds to postpone booting upon receiving power. This helps prevent HDMI detection issues when the TV and Superliminal power on at the same time.

  • Bootscreen Delay

    The time in seconds to show flyer information before launching the slideshow. This is also the interval over which a user may press any key to enter setup.

  • Overscan

    Enables or disables overscanning to better fit the slideshow to the display. This should be used if there is a black border at the edge of the display, or if the display cuts off part of the slideshow. This will require a reboot before values can be configured.

  • Overscan Values

    Amount to compress or expand the slideshow. Use positive values if a black border is visible. Use negative values if the slideshow extends beyond the screen
