Joining a Wireless Network

Superliminal can store multiple wireless networks and will automatically join a known network at boot. Networks are saved by their name, encryption, and passphrase, meaning that a wireless network can be stored without being in proximity to that network.

  • SSID

    Name (ESSID) of the network to be saved.

  • Passphrase

    The passphrase of the network to be saved. This must be greater than 8 alphanumeric characters in length.

  • Encryption

    The encryption type of the network to be joined. If you are unsure, the default of WPA2 is a good guess.

  • Use DHCP

    Toggles between dynamic (DHCP) and static aquisition of a local IP address. DHCP allows the router to automatically assign a local IP to Superliminal. Static allows the user to set a fixed local IP address. DHCP is recommended.

  • Forget Network

    Forgets a stored network configuration. Networks are listed by ESSID.
